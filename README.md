#EN
## WT AmoCRM Radicalmart message plugin for Joomla
Sending order information to your AmoCRM (aka Kommo CRM) account. Plugin for the RadicalMart component.

#RU
## WT AmoCRM Radicalmart message плагин для Joomla
Плагин умеет отправлять данные заказа из компонента интернет-магазина [RadicalMart](https://radicalmart.ru/) в [AmoCRM](https://amocrm.ru), а так же оставлять примечания к сделке при смене статуса заказа.
Для работы плагина необходима установка и настройка библиотеки [WT AmoCRM](https://github.com/WebTolk/WT-Amo-CRM-library-for-Joomla-4) для Joomla ([документация](https://web-tolk.ru/dev/biblioteki/wt-amo-crm-library/documentation)). Также библиотеку можно найти на [Github](https://github.com/WebTolk/WT-Amo-CRM-library-for-Joomla-4).
